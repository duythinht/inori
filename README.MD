## The Inori colors scheme for vim

## Install using Pathogen:

	cd ~/.vim/bundle
	git clone https://github.com/duythinht/inori

## Install using Vundle

	Plugin 'duythinht/vim-coffee'

## Modify .vimrc

	syntax enable
	colorscheme coffee
